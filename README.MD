# WikiKnowledge Graph Construction and Question Answering System

WikiKnowledge is designed to solve the problem of "fragmented" knowledge by building a hierarchical, fractal knowledge graph. Unlike Wikipedia, which can be difficult to follow as a structured educational path, WikiKnowledge uses "superstructures" (category articles) to provide summary-driven pathways through different layers of information.

See [knowledge/articles/overview.md](knowledge/articles/overview.md) for the overview of the project.
