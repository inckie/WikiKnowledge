# AI Integration

<!-- human:start -->
This category contains articles related to integrating AI agents with the WikiKnowledge system. It covers topics like the Model Context Protocol (MCP), available tools, and best practices for automated knowledge management. The goal is to enable AIs to not only query the knowledge base but also to contribute to it in a structured and meaningful way.
